MySQL1-4章总结

第一章MySQL基础

1.数据库基本概念

1.数据库

数据库就是数据表，数据库关系和数据库关系对象的集合。

2.数据库管理系统（DBMS）

数据库管理系统是一种操作和管理数据库的大型软件，用来建立，使用和维护数据库。他对数据库进行统一的管理和控制，以保证数据库的安全性和完整性；

3.数据库系统（DBS）

数据库系统是一个实际可运行的系统，是由存储介质，处理对象和管理系统组成的集合体，可以对系统提供的数据进行存储、维护和应用，它一般由软件、数据库和管理员组成；

4.数据库管理员（DBA）

负责创建数据库。监控数据库的运行，维护数据库的安全，优化数据库的性能，使数据库安全、稳定运行；

5.数据库模型

数据库模型分为层次模型、网状模型、关系模型和面向对象型；

SQL Server是一个关系数据库管理系统，两亿以内中型数据库；

Orade 2兆以内大型数据库；

Access 2万以内小型数据库

2.数据库的操作

(1)创建数据库

CREATE DATABASE 数据库名;

(2)进入数据库

USE 数据库名;

只有先进入数据库，才能进行下面的操作

(3)删除数据库

DROP DATABASE 数据库名;

(4)创建数据表

CREATE TABLE 表名(字段1 类型,字段2 类型……..);

(5)查询表的结构信息

DESC 表名;

(6)查询表的创建信息

SHOW CREATE TABLE 表名;

(7)删除表

DROP TABLE 表名;

3.修改表结构

(1)修改表名

ALTER TABLE 旧表名 RENAME 新表名;

(2)修改数据类型

ALTER TABLE 表名 MODIFY 字段名 新类型;

(3)修改字段名

ALTER TABLE 表名 CHANGE 旧字段名 新字段名 新数据类型;

(4)删除字段

ALTER TABLE 表名 DROP 字段名;

(5)添加字段

ALTER TABLE 表名 ADD 新字段名 新字段类型;

9.使用limit关键词限制行数。

位置偏移量：用于指示从哪一行开始显示，是一个可选参数，如果不指定该参数，将会从结果集的第一条记录开始(从零开始计数)。

MySQL的内置行数

1.字符串函数

CONCAT：连接字符串；

select concat(‘今天’,’晴天’) 返回：今天晴天；

LENGTH:返回字符串的长度；

length(‘hello word’) 结果：11；

注意：一个字代表三个字符；length(‘mysql数据库’)结果为：14;

UPPER:将字符串转为大写；

upper(‘abcd’) 返回：ABCD；

LTRIM:去除左边的空格；

ltrim(‘ abc’) 返回：’abc’;

RTRIM:去除右边的空格；

rtrim(‘abc ‘) 返回：’abc’;

TRIM:去除左右两边的空格;

trim(‘ ABC ‘) 返回’ABC’;

REPLACE:替换；

replace(‘abcccd’,’c’,’x’) 将字符串中的c替换为x; 返回：’abxxxd’;

STRCMP:比较字符串;

strcmp(s1,s2);

SUBSTRING:截取字符串；

substring(‘johnson’,5,3) 从第五个位置开始截取长度为3的字符串。返回：’son’;

2.日期函数

获得当前日期+时间(date+time)函数: now();

获得当前日期函数：CURDATE();

获得当前时间函数：CURTIME();

返回日期对应的工作日索引：WEEKDAY(DATE);

0表示周一，1表示周二，2表示周三，3表示周四，4表示周五，5表示周六，6表示周日；

返回日期对应的一周中的索引：DAYOFWEEK(DATE);

1表示周日，2表示周一，3表示周二，4表示周三，5表示周四，6表示周五，7表示周六；

3.系统函数

DATABASE();返回当前数据库名；

BENCHMARK(count,expr);将表达式expr重复运行count次；

CONNECTION_ID();返回当前客户的ID；

FOUND_ROWS();返回最后一个select查询进行检索的总行数；

VERSION();返回MySQL服务器的版本；

例如：返回当前的数据库名和版本；
