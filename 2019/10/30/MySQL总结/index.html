第一章MySQL基础

1.数据库基本概念

1.数据库

数据库就是数据表，数据库关系和数据库关系对象的集合。

2.数据库管理系统（DBMS）

数据库管理系统是一种操作和管理数据库的大型软件，用来建立，使用和维护数据库。他对数据库进行统一的管理和控制，以保证数据库的安全性和完整性；

3.数据库系统（DBS）

数据库系统是一个实际可运行的系统，是由存储介质，处理对象和管理系统组成的集合体，可以对系统提供的数据进行存储、维护和应用，它一般由软件、数据库和管理员组成；

4.数据库管理员（DBA）

负责创建数据库。监控数据库的运行，维护数据库的安全，优化数据库的性能，使数据库安全、稳定运行；

5.数据库模型

数据库模型分为层次模型、网状模型、关系模型和面向对象型；

SQL Server是一个关系数据库管理系统，两亿以内中型数据库；

Orade 2兆以内大型数据库；

Access 2万以内小型数据库

2.数据库的操作

(1)创建数据库

CREATE DATABASE 数据库名;

(2)进入数据库

USE 数据库名;

只有先进入数据库，才能进行下面的操作

(3)删除数据库

DROP DATABASE 数据库名;

(4)创建数据表

CREATE TABLE 表名(字段1 类型,字段2 类型……..);

(5)查询表的结构信息

DESC 表名;

(6)查询表的创建信息

SHOW CREATE TABLE 表名;

(7)删除表

DROP TABLE 表名;

3.修改表结构

(1)修改表名

ALTER TABLE 旧表名 RENAME 新表名;

(2)修改数据类型

ALTER TABLE 表名 MODIFY 字段名 新类型;

(3)修改字段名

ALTER TABLE 表名 CHANGE 旧字段名 新字段名 新数据类型;

(4)删除字段

ALTER TABLE 表名 DROP 字段名;

(5)添加字段

ALTER TABLE 表名 ADD 新字段名 新字段类型;

MySQL的内置行数

1.字符串函数
CONCAT：连接字符串；

TRIM:去除左右两边的空格;

trim(‘ ABC ‘) 返回’ABC’;

REPLACE:替换；

STRCMP:比较字符串;

strcmp(s1,s2);

SUBSTRING:截取字符串；

substring(‘johnson’,5,3) 从第五个位置开始截取长度为3的字符串。返回：’son’;

第六章多表连接查询

1.连接查询

常用的连接查询有：内连接，外连接，交叉连接；

1.交叉连接是将多张表中的数据一一对应连接在一起，形成结果集的数据相当于多张表的数据行数“相乘”之后的结果。

交叉表中没有where字句；

例如：将T_A表名改成A表，把T_B表名改成B表，并查询两个表中的所有信息；

实际上，select语句中可以实现两张表以上的表的连接查询，其创建规则与两个表连接相同，只需要在where条件中增加一个AND复合条件；

例如：三张表，学生姓名存放于studentinfo表中，科目名称存放于subject表中，考试成绩存放于exam表中
